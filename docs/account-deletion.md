# 오늘도수영 계정 삭제 안내

시행일: 2026-06-30

오늘도수영 계정은 앱 안에서 직접 삭제할 수 있습니다.

## 앱에서 삭제하는 방법

1. 오늘도수영 앱에 로그인합니다.
2. 하단 탭에서 `내정보`를 엽니다.
3. `계정 삭제`를 누릅니다.
4. 안내 내용을 확인하고 삭제를 확정합니다.

계정 삭제가 완료되면 로그인 계정과 연결된 회원 정보, 예약 정보, 결석 정보, 특별수업 신청 정보, 횟수권 이력이 삭제됩니다.

## 앱에 로그인할 수 없는 경우

앱에 로그인할 수 없거나 별도 삭제 요청이 필요한 경우 아래 연락처로 요청하세요.

- 전화: 010-4698-3505

요청 시 계정을 확인할 수 있는 이메일 주소와 휴대폰 번호를 알려주세요. 본인 확인 후 계정 삭제를 진행합니다.
