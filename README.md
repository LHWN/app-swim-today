# 오늘도수영

iOS 우선 테스트를 목표로 만든 Expo/React Native 모바일 앱입니다. 같은 코드베이스에서 Android 빌드로 확장할 수 있습니다.

## 실행

```bash
npm install
npm run ios
```

iOS Simulator 접근이 어려운 환경에서는 웹 미리보기로 기능 흐름을 확인할 수 있습니다.

```bash
npm run web
```

## 포함 기능

- 평일 06:00-23:00, 주말 09:00-18:00 한 시간 단위 수업 예약
- 한 타임당 한 회원만 확정 예약, 이미 예약된 시간은 대기 등록
- 회원별 횟수권 잔여 횟수 확인 및 예약/취소 시 반영
- 관리자 홈에서 회원별 확정 예약과 대기 인원 확인
- 네이버, 카카오, 구글, 애플 SSO 가입 UI
- 관리자 전용 공지 작성 및 이미지 첨부
- 수업 임박, 예약 확정, 공지사항, 출석 리마인드, 강사/일정 변경, 재예약 추천 알림 설정
- 문의 전화 `010-4698-3505`

실제 SSO, 관리자 권한, 푸시 발송은 운영 백엔드와 각 플랫폼 개발자 콘솔 설정이 필요합니다. 현재 버전은 로컬 테스트용 MVP입니다.
